## 🎯 关键特性

### 1. 状态标识（一目了然）
- **🟢 正常**：使用率 < 80%，绿色标签
- **🟠 即将用尽**：使用率 >= 80%，橙色标签
- **🔴 已用尽**：剩余额度 = 0，红色标签

### 2. 进度条颜色随状态变化
- 正常：蓝色渐变（#007AFF → #5856D6）
- 警告：橙色渐变（#FF9500 → #FF6B00）
- 危险：红色渐变（#FF3B30 → #D32F2F）

### 3. 交互效果
- **Hover卡片**：向上浮动 + 边框高亮
- **选中卡片**：蓝色边框 + 浅蓝背景
- **复制成功**：按钮变绿 + Toast提示
- **操作按钮**：Hover时向上浮动

### 4. 视图切换
```
[📇 卡片视图] [📊 表格视图]  每页显示 [10条▼]
     ↑ 激活                              
```

## 📱 响应式适配

### 超宽屏（> 1400px）
- 卡片：4-5列
- 最小宽度：380px
- 间距：24px

### 桌面端（900px - 1400px）
- 卡片：2-3列  
- 最小宽度：380px
- 间距：24px

### 平板竖屏（768px - 900px）
- 卡片：2列
- 最小宽度：320px
- 间距：16px

### 手机（< 768px）
- 卡片：1列
- 全宽显示
- 间距：12px

## 🚀 性能优化

1. **CSS动画使用GPU加速**
   - transform (translateY, scale)
   - opacity

2. **只渲染当前页**
   - 支持分页（10/30/100/全部）
   - 减少DOM节点数量

3. **缓存机制**
   - localStorage缓存API Keys
   - 减少网络请求

4. **并发控制**
   - 限制同时8个请求
   - 避免服务器压力

## 🎨 设计亮点

### Apple风格设计
- 圆角：12px-24px
- 阴影：柔和多层
- 字体：SF Pro Display风格
- 动画：平滑 cubic-bezier

### 信息层次清晰
1. **第一眼**：状态标签（正常/警告/危险）
2. **第二眼**：进度条（直观百分比）
3. **第三眼**：统计数字（详细数据）
4. **辅助信息**：时间、ID等

### 颜色系统
```css

--color-success: #34C759;   /* iOS绿 */
--color-warning: #FF9500;   /* iOS橙 */

--color-secondary: #5856D6; /* iOS紫 */
```

## 🔧 技术栈

- **前端**：纯HTML + CSS + JavaScript
- **后端**：Deno + Deno KV
- **字体**：Fira Code（等宽字体）
- **图标**：Emoji + SVG

## 💡 使用建议

### 推荐使用卡片视图的场景：
- ✅ 竖屏显示器或窄窗口
- ✅ 快速浏览余额状态
- ✅ 需要关注单个Key详情
- ✅ 移动设备访问

### 推荐使用表格视图的场景：
- ✅ 横屏大屏幕
- ✅ 需要快速对比多个Key
- ✅ 需要看到更多Key（一屏显示更多）
- ✅ 批量操作前的筛选
